Tecnologías:

HTML

JavaScript

CSS

Node.js(Servidor local)

MySQL

Ngrok (VPS)

Características:

Interfaz web responsiva.

Consumo de una API REST para mostrar productos.

Gestión de base de datos con MySQL.
